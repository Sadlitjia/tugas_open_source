- Tentukan Lokasi Instalasi Git
- Pilih Komponen Tambahan
- Tentukan Nama Aplikasi Git
- Tentukan File Editor
- Atur Path Environment
- Pilih Aplikasi SSH
- Pilih Line Ending
- Pilih Emulator Terminal
- Tentukan Opsi ekstra
- Mulai Proses Instalasi
- Cek Versi Git
