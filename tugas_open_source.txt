Mengunduh dan menginstal GitHub Desktop
- Buka browser di laptop Anda
- Kunjungi halaman unduhan untuk GitHub Desktop.
- Klik Unduh untuk macOS.
- Di folder Unduhan komputer Anda, klik dua kali file zip GitHub Desktop.
- Setelah file di-unzip, klik dua kali GitHub Desktop.
- Desktop GitHub akan diluncurkan setelah penginstalan selesai.
- Aplikasi sudah bisa digunakan selama terhubung ke internet
